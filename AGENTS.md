# AGENTS

Repository conventions for coding agents working in this project.

## Stack

- VS Code extension using TypeScript + ESM modules.
- Node.js 24.x runtime.
- TextMate grammar injection generated by script.

## Package manager

- Use `pnpm` for commands.
- If `pnpm` is blocked by strict package-manager checks in this repo, use:
  - `COREPACK_ENABLE_STRICT=0 pnpm <command>`

## Build and quality

- Run formatting/lint checks with:
  - `pnpm run biome`
- Compile with:
  - `pnpm run compile`
- Build grammar from source script with:
  - `node ./scripts/build-grammar.mjs`

## Grammar workflow

- Source of truth for grammar generation is:
  - `scripts/build-grammar.mjs`
- Generated output is:
  - `syntaxes/syntax-highlight-comment.tmLanguage.json`
- If grammar behavior changes, update the generator and regenerate the JSON.

## Code conventions

- Use ESM imports and prefer `node:`-prefixed built-in Node imports.
- Keep edits minimal and focused on the requested behavior.
- Keep diagnostics and marker aliases in sync when adding/removing marker types.
- Update `CHANGELOG.md` for user-visible behavior changes.
